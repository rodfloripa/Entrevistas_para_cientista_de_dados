# Entrevistas

Perguntas e repostas sobre ciência de dados
<p align="justify">
Processos seletivos pelo Linkedin estão bem concorridos,sendo difícil encontrar uma vaga com menos de
100 inscritos. Em geral 60% das pessoas que se inscrevem são de nível júnior, então a concorrência real
para vagas de pleno e senior é de 40%. O intuito deste reposiório é auxiliar aqueles que estão buscando
uma vaga. Os desafios são todos de empresas brasileiras.
</p>
